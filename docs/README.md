# SAP BTP DevOps Workshop

This is a Hands-On Workshop designed to bridge the gap between **writing code** and **running enterprise-grade operations**. Moving past manual deployments and diving deep into the core tools that power modern SAP landscapes.

In this session, we'll build a full automated pipeline from scratch, covering:

- Cloud Foundry Runtime - Setting up your foundation.

- SAP Business Application Studio - Developing with a cloud-native mindset.

- SAP Cloud Transport Management - Managing professional release cycles.

- SAP Continuous Integration and Delivery - Automating the "push-to-prod" flow.

Whether you're a developer looking to automate your workflow or an architect defining the next big project, this workshop is designed to give you the practical, **click-by-click** experience needed to master the SAP BTP DevOps ecosystem.

## Prerequisites

- You have an account on SAP Business Technology Platform. See [Enterprise Accounts](https://help.sap.com/viewer/65de2977205c403bbc107264b8eccf4b/Cloud/en-US/046f127f2a614438b616ccfc575fdb16.html).

- You're an Administrator of your global account and Org Manager of your subaccount on SAP BTP. See [Administrator Role Collections](https://help.sap.com/docs/btp/sap-business-technology-platform/role-collections-and-roles-in-global-accounts-directories-and-subaccounts?locale=en-US#administrator-role-collections)

- You have a GitHub account. See [Creating an account on GitHub](https://docs.github.com/en/get-started/start-your-journey/creating-an-account-on-github).

- You have cloned the workshop project: `https://github.com/mauriciolauffer/btpdevops.git`. See [Cloning a repository](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository).
